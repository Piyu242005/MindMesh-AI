# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [v1.0.0] - Initial Release

### Added
- Automated Video Processing pipeline
- Faster-Whisper integration for robust, local audio transcription
- Intelligent Chunking & Embedding using Sentence Transformers (`BAAI/bge-small-en-v1.5`)
- Qdrant Cloud integration for scalable, high-performance vector search
- Smart LLM Routing supporting Google Gemini and Groq
- Streamlit UI including an Upload Center, Settings, Dashboard, and AI Chat interface
- Full rebranding to "MindMesh AI"
