# Support & Getting Help

Welcome to the MindMesh AI community! If you need help with setting up, using, or contributing to the project, here is how you can get support:

## 1. Check the Documentation
Before opening an issue, please review our comprehensive documentation:
- [README.md](./README.md) - For installation and basic setup.
- [ARCHITECTURE.md](./ARCHITECTURE.md) - To understand how the platform works under the hood.
- [DEPLOYMENT.md](./DEPLOYMENT.md) - For cloud deployment guides.

## 2. GitHub Issues (Bug Reports & Feature Requests)
We use GitHub Issues to track bugs and feature requests. 

* **Bug Reports**: If you've found a bug, please open an issue using the **Bug Report template**. Provide as much detail as possible, including logs, screenshots, and steps to reproduce.
* **Feature Requests**: If you have an idea for a new feature, open an issue using the **Feature Request template**. We love community suggestions!

[Open a new issue here](https://github.com/Piyu242005/MindMesh-AI/issues)

## 3. GitHub Discussions
For general questions, troubleshooting help, or sharing what you've built with MindMesh AI, please use GitHub Discussions. It's a great place to connect with other users and contributors.

* **Q&A**: Ask the community for help with your specific setup.
* **Show and Tell**: Show off your deployed MindMesh AI instance or custom modifications!

[Join the Discussions here](https://github.com/Piyu242005/MindMesh-AI/discussions)

## 4. Contact the Maintainer
If you have a business inquiry, security report, or need private support, you can reach out directly to the author, **Piyush Ramteke**.
